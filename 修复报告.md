- **验证**：登录后 JWT 含 `"role":"ADMIN"`，全部管理 API 返回 200

## 二、严重安全缺陷修复（S2-S4）

### ✅ S2 — 会员充值无支付校验
- **文件**：`UserServiceImpl.java`, `UserService.java`, `UserController.java`
- **问题**：`rechargeMembership()` 无支付验证，任何人调用 API 即可免费获取 VIP
- **修复**：
  1. 新增 `@Value("${music.membership.skip-payment:true}")` 配置开关
  2. 方法签名增加 `paymentId` 参数
  3. 生产模式（`skip-payment=false`）时校验 `paymentId` 非空
  4. 接口文档标注开发/生产行为差异
- **验证**：开发模式（默认）行为不变；生产模式配置后生效

### ✅ S3 — 文件上传路径硬编码
- **文件**：`FileServiceImpl.java`
- **问题**：`UPLOAD_DIR = "E:/07_Data/IdeaProjects/music/data/music"` 硬编码 Windows 绝对路径
- **修复**：
  1. 改为 `@Value("${music.file.upload-dir:data/music}")` 注入
  2. 增加 @PostConstruct 自动转换为绝对路径
  3. 启动日志输出："文件上传根目录: xxx"
- **验证**：启动日志显示正确路径 `E:\...\data\music`

### ✅ S4 — 静态资源路径硬编码
- **文件**：`WebMvcConfig.java`
- **问题**：`UPLOAD_PATH = "file:E:/.../data/music/"` 硬编码
- **修复**：
  1. 改为 `@Value("${music.file.upload-dir:data/music}")` 注入
  2. @PostConstruct 自动解析绝对路径
  3. `addResourceHandlers` 使用动态 `uploadPath`
- **验证**：启动日志显示"静态资源映射目录: xxx"

## 三、高风险缺陷修复（H1-H5）

### ✅ H1 — 批量导入无事务
- **文件**：`AdminContentServiceImpl.java`
- **修复**：`batchImportSongs()` 添加 `@Transactional(rollbackFor = Exception.class)`
- **效果**：部分歌曲导入失败时自动回滚全部

### ✅ H2 — 看板播放量数据模拟
- **文件**：`AdminDashboardServiceImpl.java`, `PlayHistoryMapper.java`, `PlayHistoryMapper.xml`
- **问题**：使用正弦波模拟每日播放量，非真实数据
- **修复**：
  1. `PlayHistoryMapper` 新增 `countByDate(LocalDate date)` 方法
  2. `PlayHistoryMapper.xml` 新增 `SELECT COUNT(*) FROM play_history WHERE DATE(played_time) = #{date}`
  3. DashboardServiceImpl 注入 PlayHistoryMapper 替代正弦波计算
- **效果**：每日播放量数据来自 play_history 表真实记录

### ✅ H3 — SHUFFLE 模式损坏
- **文件**：`frontend/src/stores/playback.js`
- **问题**：`playedSet` 从未被填充，shuffle 模式下歌曲可能重复播放
- **修复**：在 `play()` 函数中，当模式为 SHUFFLE 时将当前 `queueIndex` 加入 `playedSet`
- **效果**：Shuffle 模式下所有歌曲播完一轮前不会重复

### ✅ H4 — 搜索高亮显示为纯文本
- **文件**：`frontend/src/views/Search.vue`
- **问题**：`{{ highlight(text, keyword) }}` 使用文本插值导致 `<mark>` 标签被转义
- **修复**：改为 `<span v-html="highlight(...)"></span>`
- **效果**：搜索关键词正确高亮显示

### ✅ H5 — PlayQueue SVG 编码错误
- **文件**：`frontend/src/components/PlayQueue.vue`
- **问题**：SVG data URI 中 `<` `>` 未编码
- **修复**：使用 `encodeURIComponent()` 编码 SVG 字符串
- **效果**：封面加载失败时显示正确的默认图标

## 四、中等风险缺陷修复（M1-M11）

### ✅ M1-M3 — 事务外发问题（3 处）
- **文件**：`NotificationServiceImpl.java`, `AdminNoticeServiceImpl.java`, `AdminReportServiceImpl.java`
- **问题**：Redis Pub/Sub 发布在 `@Transactional` 内部，事务回滚后用户仍收到通知
- **修复**：使用 `TransactionSynchronizationManager.registerSynchronization()` 将 Redis 发布延迟到 `afterCommit()`
- **效果**：事务提交成功后才推送消息，回滚后不发送

### ✅ M4 — 播放计数竞态条件
- **文件**：`PlayCountFlushTask.java`
- **问题**：Redis GET 后 DEL 之间新增播放量可能丢失
- **修复**：使用 `StringRedisTemplate.opsForValue().getAndDelete(key)`（GETDEL 命令）原子读取并删除
- **效果**：无竞态窗口，零数据丢失

### ✅ M5 — 房间操作缺事务
- **文件**：`RoomServiceImpl.java`
- **修复**：`joinRoom()`, `leaveRoom()`, `kickMember()`, `transferOwner()` 添加 `@Transactional`
- **效果**：Redis 与 MySQL 操作保持事务一致性

### ✅ M6 — WebSocket 未检查 Token 黑名单
- **文件**：`NotifyWebSocketInterceptor.java`, `RoomWebSocketInterceptor.java`
- **问题**：两个 WS 拦截器未检查 JWT 是否在黑名单中
- **修复**：注入 `CacheUtil`，在 token 校验后增加 `cacheUtil.exists(RedisKeys.BL_TOKEN + jti)` 检查
- **效果**：已登出用户无法建立 WS 连接

### ✅ M7 — WebSocket 未限制帧大小
- **文件**：`WebSocketConfig.java`
- **问题**：未配置 `maxTextMessageBufferSize`/`maxBinaryMessageBufferSize`
- **修复**：新增 `ServletServerContainerFactoryBean` Bean，配置 64KB 帧大小限制
- **效果**：防止超大消息导致 OOM

### ✅ M8 — Chart.js 实例未清理
- **文件**：`frontend/src/views/AdminDashboard.vue`
- **修复**：`onUnmounted()` 中调用 `userChart.destroy()` 和 `genreChart.destroy()`
- **效果**：离开管理后台页面时释放图表内存，无泄漏

### ✅ M9 — 密码校验规则不一致
- **文件**：`frontend/src/views/Profile.vue`, `frontend/src/views/Register.vue`
- **问题**：注册要求 8-20 位含字母数字，个人资料只要求 6 位
- **修复**：Profile.vue 改为与 Register.vue 一致的校验规则（8-20 位，含字母和数字）
- **效果**：统一密码强度标准

### ✅ M10 — /admin 路由缺管理员守卫
- **文件**：`frontend/src/router/index.js`
- **问题**：已登录非管理员可看到后台布局
- **修复**：添加 `meta: { requiresAdmin: true }` + `router.beforeEach` 中校验 `authStore.isAdmin`
- **效果**：非管理员访问管理后台自动重定向到首页

### ✅ M11 — IME 组合状态冲突
- **文件**：`frontend/src/components/CommentSection.vue`
- **问题**：中文输入法下回车误提交
- **修复**：添加 `onReplyKeydown()` 处理器，检查 `event.isComposing || event.keyCode === 229`
- **效果**：IME 组合期间回车不提交

### ✅ L8 — 前端 import 位置错误
- **文件**：`frontend/src/components/CommentSection.vue`

### ✅ L10 — PlayerBar 移动端适配

- **修复**：添加 `@media (max-width: 768px)` CSS，隐藏封面缩略图、缩小按钮、隐藏音量条
